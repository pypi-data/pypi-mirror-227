### BIP38

[![PyPI Version](https://img.shields.io/pypi/v/bip38.svg?color=blue)](https://pypi.org/project/bip38)

A python library for implementation of Bitcoin Improvement Proposal (BIP38) protocol.

#### Installation

```
pip install bip38
```
